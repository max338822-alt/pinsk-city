# Пинск — Городской портал

🏙️ Городской портал Пинска / Pinsk city portal — Next.js 16 + React 19 + Tailwind CSS

## О проекте

Современный информационный портал города Пинска (Брестская область, Беларусь).
Новости, кибербезопасность, технологии, события и городская жизнь.

## Стек

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion

## Запуск

```bash
npm install
